# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

Güvenlik açığı bulduysanız, lütfen **hamzaince001@gmail.com** adresine e-posta gönderin.

**Önemli:** Güvenlik açıklarını public issue olarak açmayın. E-posta ile bildirin.

Güvenlik açığını bildirdiğinizde:
1. Açığın detaylı açıklaması
2. Etkilenen versiyonlar
3. Potansiyel etki
4. Önerilen çözüm (varsa)

Bildirdiğiniz güvenlik açığı için teşekkür ederiz!
